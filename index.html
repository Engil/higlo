Higlo

Higlo is an OCaml library for syntax highlighting. Higlo could mean Highlighting in Ocaml.

The purpose of Higlo is not to provide syntax highlighting for every language, nor target every format (HTML, LaTeX, ...). It provides a simple way to support additional languages and develop the generator for the output format you need.

By now, only OCaml and XML input languages are provided, and it only generates Xtmpl.tree to highlight code in XML. Developing support for additional input languages is just a matter of writing a Ulex lexer. Supporting another output format only requires mapping values of type token.

See the OCaml lexer for an example of Higlo Ulex lexer.

Installation

Higlo requires Xtmpl >= 0.8 and Ulex >= 1.1 to be installed.

To compile and install:

  • tar xvfz higlo-XXX.tar.gz
  • cd higlo-XXX
  • make all install

This will install the higlo and higlo.lexers packages with findlib.
