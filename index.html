Higlo

Higlo is an OCaml library for syntax highlighting. Higlo could mean Highlighting in Ocaml.

Download

Higlo is hosted on Github.

Releases
  • 0.1 [2014-03-05]: Initial release.
Installation

Higlo requires Xtmpl >= 0.8 and Ulex >= 1.1 to be installed.

To compile and install:

  • tar xvfz higlo-XXX.tar.gz
  • cd higlo-XXX
  • make all install

This will install the higlo and higlo.lexers packages with findlib.
